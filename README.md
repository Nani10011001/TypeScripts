# 🚀 Full Stack TypeScript Project (Node.js + Express + React)

A modern full-stack application built using **TypeScript** with a clean architecture separating backend and frontend.

---

## 📂 Project Structure

```
.
├── node-ts-express   # Backend (Node.js + Express + TypeScript)
├── react-ts          # Frontend (React + TypeScript + Vite)
├── TypeScript        # Practice / learning TypeScript modules
```

---

## ⚙️ Tech Stack

### 🖥️ Backend

* Node.js
* Express.js
* TypeScript
* ts-node / ts-node-dev

### 🌐 Frontend

* React
* TypeScript
* Vite
* ESLint

### 📦 Tools & Libraries

* Zod (Validation)
* dotenv (Environment Variables)
* Nodemon / ts-node-dev (Development)

---

## 🚀 Features

* ✅ Type-safe backend using TypeScript
* ✅ REST API with Express
* ✅ React frontend with Vite
* ✅ Input validation using Zod
* ✅ Environment variable validation
* ✅ Modular and scalable folder structure
* ✅ Separate frontend & backend architecture

## 📦 Installation

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

---

### 2️⃣ Backend Setup

```bash
cd node-ts-express
npm install
```

Run backend:

```bash
npm run dev
```

Build:

```bash
npm run build
```

---

### 3️⃣ Frontend Setup

```bash
cd ../react-ts
npm install
npm run dev
```

---

## 🔐 Environment Variables

Create a `.env` file in backend:

```
PORT=5000
```

(Optional: Add validation using Zod)

---

## 📌 Scripts

### Backend

* `npm run dev` → Run development server
* `npm run build` → Compile TypeScript
* `npm start` → Run compiled code

### Frontend

* `npm run dev` → Start Vite server
* `npm run build` → Build production app
